You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /round -- Desrciption: Rounds floating point values to whole numbers. Params:num1=(float) num2=(float)
    ===>Example URL: /round?num1=3&num2=3
  • /build -- Desrciption: Builds two strings together. Params:part1=(String) part2=(String)
    ===>Example URL: /build?part1=My name is Shaun and i love &part2=SER321
